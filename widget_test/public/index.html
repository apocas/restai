Widget Context Injection Test

Generate signed JWT tokens and test the RESTai widget context feature

How it works:
1. Enter your Widget Key and Context Secret (from the Widget tab in your project)
2. Define context claims (e.g. user name, plan, account ID) as JSON
3. Click Generate Token & Load Widget to sign a JWT and inject the widget
4. The widget sends X-Widget-Context header — the server verifies it and injects claims into the system prompt
5. Use {{context.key}} template variables in your project's system prompt to reference the claims

Configuration

The base URL of your RESTai instance

Shown once when you click "Generate Context Secret" in the Widget tab
These claims will be available as {{context.user_name}}, {{context.plan}}, etc. in the system prompt
How long the token is valid (max 86400 = 24h)
