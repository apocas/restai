Features

One plugin, every AI surface in WordPress

Each capability is wired through to a dedicated RESTai project. Auto-provisioned on first connect, fully editable from your RESTai admin afterwards.

✍️

Content writer

Generate post bodies and excerpts straight from Gutenberg. Optional brief, replaces or appends to the editor with one click.

🔎

SEO assistant

Title, meta description, focus keyphrase and tag suggestions in JSON. Writes directly into Yoast and Rank Math fields.

🖼️

Featured image generation

Calls whichever image generator your team has access to (Flux, SDXL, DALL·E…). Auto-attaches as the featured media.

🌍

Translations

Translate any post into N languages, saved as drafts. Polylang/WPML aware so the language gets attached automatically.

💬

Comment moderation

Pre-publish AI moderation flags spam and toxic content, suggests replies for your team — before anything hits your site.

🛒

WooCommerce

Product description generator and FAQ shortcode driven by your catalog data. Cached so repeat visitors don't re-bill.

📚

Knowledge sync

Every published post and page is auto-pushed into a Support Bot RAG project, so the bot is always current.

🧠

AI site search

Replace native WordPress search with semantic answers from the support bot — citing your own pages back to visitors.

💭

Embeddable chat widget

One toggle adds a chat bubble to public pages. Plugin auto-creates the right widget key, allow-lists your domain — no copy/paste needed.

📨

Email personalization

Welcome, password reset, abandoned cart — rewritten to feel personal and on-brand via the wp_mail filter.

📊

Analytics in WP admin

Token usage, cost and request volume mirrored into your WP dashboard. No need to leave to know what AI is actually spending.

🤖

Image alt text

Optional auto alt text on uploads using a vision model — accessibility compliance without thinking about it.

How it works

Up and running in three steps

Install the plugin

Upload the plugin zip via Plugins → Add New → Upload Plugin, or drop the folder into wp-content/plugins/restai and activate.

Connect to your RESTai

In Settings → RESTai, paste your RESTai URL + API key, click Test connection, pick a team from the dropdown, save.

Auto-provision projects

Click Auto-provision starter projects. The plugin creates Content Writer, SEO Assistant, Translator, Support Bot, and 4 others on your RESTai. Re-map any to a different project later.

Auto-provisioned

Eight RESTai projects, ready on day one

Each task the plugin performs is one RESTai project — so you can swap models per capability, set per-project budgets, audit usage independently, version prompts, and re-use the same projects from the API or other clients.

  • wp-content-writer — Gutenberg / classic editor body and excerpts
  • wp-seo-assistant — JSON-only SEO meta + tag suggestions
  • wp-translator — HTML-preserving translations to N languages
  • wp-comment-moderator — spam / toxicity / suggested replies
  • wp-support-bot (RAG) — knowledge synced from posts & pages
  • wp-product-writer — WooCommerce descriptions and FAQs
  • wp-email-personalizer — rewrites transactional emails
  • wp-image-alt — vision-model alt text for new uploads
Install

Three ways to install

From WordPress admin (recommended)
# 1. Download the latest release zip
curl -LO https://github.com/apocas/restai/releases/latest/download/restai-wordpress.zip

# 2. Plugins → Add New → Upload Plugin → choose zip → Install Now → Activate
From source (development)
# Clone the RESTai repo
git clone https://github.com/apocas/restai
cp -r restai/wordpress/restai \
      /path/to/wp-content/plugins/restai

# Then activate from the admin UI
With WP-CLI
# Build a zip and install in one shot
cd restai/wordpress && zip -r restai.zip restai
wp plugin install restai.zip --activate

You own the data

Calls go from your WordPress to your RESTai instance. No third-party SaaS in between, no telemetry from the plugin.

Per-task budgets

Each WP capability is a separate RESTai project — set rate limits and budgets per project to cap spend on any one feature.

Built for WP

Sanitised input, escaped output, capability checks, nonces, i18n-ready. GPL-2.0+ — same model as WordPress itself.
