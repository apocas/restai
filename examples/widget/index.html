RESTai chat widget

Add an AI chat bubble to any website with a single <script> tag — no build step, no frontend framework. The widget streams responses, keeps conversation context, and isolates its styles with a Shadow DOM. The bubble for this page is in the bottom-right corner. 👉

Before this works, replace the two placeholders in the script tag below:
  • data-project-id — the integer id of the project to chat with.
  • data-api-key — a read-only, project-scoped key. Generate one from the project's Widget tab in /admin (it's safe to expose in page source — same model as a Stripe publishable key).
Also set the src and data-server to your RESTai server's URL.

The embed

<script
  src="https://your-restai.com/widget/chat.js"
  data-server="https://your-restai.com"
  data-project-id="1"
  data-api-key="sk-your-readonly-widget-key"
  data-title="Support Bot"
  data-subtitle="Ask me anything"
  data-welcome-message="Hi! How can I help?"
  data-primary-color="#6366f1"
  data-position="right"
  data-stream="true"
></script>

Every knob is a data-* attribute: data-title, data-subtitle, data-primary-color, data-text-color, data-position (left/right), data-welcome-message, data-avatar-url, and data-stream. Preview them live in the project's Widget tab before deploying.
